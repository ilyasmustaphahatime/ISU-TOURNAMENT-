How Organizers Use The Website Day By Day

Think of the website as your tournament control room. You set up teams first, build the fixtures, update the results when games finish, then add the player stats that power the league table, highlights, cards, awards, and news.

Best Working Order
  1. Add teams
  2. Add players
  3. Create fixtures
  4. Update results
  5. Enter player stats
  6. Publish awards and news

Quick Flow

If you only remember one thing, follow this order every time.

1

Create The Teams

In Manage Teams, add the team number, team name, and captain name. If a team already exists, load it first, then update it.

2

Add The Players

In Manage Players, use the team number and jersey number together. That pair is what tells the system which player you are saving.

3

Build The Fixtures

In Manage Matches, create every match with a unique match ID, set the date and time, and mark it as scheduled.

4

Save The Result

When the match ends, save the same match ID again, change the status to played, and enter the final score.

5

Add Player Stats

In Manage Player Stats, add goals, assists, cards, and clean sheets. These numbers feed the leaderboards and highlights.

6

Publish The Story

Open News Hub to publish weekly news, set spotlight awards, and keep the public side of the tournament fresh.

Before You Start

A small checklist that prevents most organizer mistakes.

Make sure the team numbers are correct

Everything connects through team numbers. If the team number is wrong, the players, fixtures, and stats will point to the wrong club.

Make sure the jersey number is correct

Player stats are saved using team number + jersey number. A wrong jersey number means the stats can land on the wrong player.

Create the match before adding stats

The stats form needs a real match ID. If the match does not exist yet, the stat line cannot be connected properly.

Using Manage Teams

This section is for clubs and captain information only.

Add A New Team

  1. Open Manage Teams.
  2. Enter Team Number.
  3. Enter Team Name.
  4. Enter Captain Name.
  5. Click Add New Team.

Update An Old Team

  1. Enter the number in Load Existing Team Number.
  2. Click Load Team.
  3. Edit the team name or captain name.
  4. Click Update Loaded Team.

Delete A Team

  1. Enter the number in Delete Team Number.
  2. Click Delete Team.
  3. Use this carefully, because players and related data depend on teams.

Using Manage Players

This is where organizers build each team roster.

Add Or Update A Player

  1. Enter the Jersey Number.
  2. Enter the Player Name.
  3. Enter the Position.
  4. Enter the Player Team Number.
  5. Click Save Player.

If the same team number and jersey number already exist, the system updates that player instead of creating a second one.

Delete A Player

  1. Enter the Delete Player Team Number.
  2. Enter the Delete Jersey Number.
  3. Click Delete Player.

Fixtures And Match Workflow

This is the most important organizer flow, because the fixtures and final scores drive the whole tournament table.

Before The Match

Create The Fixture

Go to Manage Matches and enter:

  • Match ID: a unique number for that game
  • Status: set it to scheduled
  • Match Date and Match Time
  • Home Team Number and Away Team Number
  • Referee ID if you use referees
  • Leave goals as 0 if the game has not started yet

Then click Save Match.

When The Match Ends

Update The Result

Use the same Match ID again and update the existing fixture:

  • Change Status to played
  • Enter the final Home Goals
  • Enter the final Away Goals

Click Save Match again. This is what updates the League Table.

If Plans Change

Postpone Or Cancel

If the game does not happen on time, use the same match and change the status to:

  • postponed if it will happen later
  • cancelled if it is removed completely

Using Manage Player Stats

This section powers the leaderboard highlights and player records.

What To Enter

  • Match ID
  • Player Team Number
  • Jersey Number
  • State (usually played)
  • Goals
  • Assists
  • Yellow Cards
  • Red Cards
  • Clean Sheets

What Changes Automatically

  • Top Scorers changes when you add goals
  • Top Assists changes when you add assists
  • Clean Sheets changes when you add clean sheets
  • Yellow Cards and Red Cards change from the card counts
  • Leaderboard Highlights reads from this live data

Delete A Wrong Stat Line

  1. Enter the Delete Match ID.
  2. Enter the Delete Player Team Number.
  3. Enter the Delete Jersey Number.
  4. Click Delete Stat Line.

News Hub And Awards

Use this part when you want the website to feel active, official, and public-facing.

Weekly News

Open News Hub and use Publish Weekly News to write a human summary of what happened during the week.

You can leave News ID empty to create a new post, or reuse an old ID to update an existing story.

Spotlight Awards

Use Manage Spotlight Awards to set things like:

  • Best Player Of The Week
  • Man Of The Match
  • Best Goalkeeper
  • Best Team Of The Week
  • Best Team Of The Month

Common Mistakes To Avoid

These are the mistakes that usually make organizers think the website is wrong, when it is really a data order issue.

Adding stats before the match exists

Create the match first in Manage Matches, then add player stats.

Using the wrong team number

The system uses team numbers everywhere. Always confirm the team number before saving players, fixtures, or awards.

Using the wrong jersey number

Stats and player updates depend on the exact jersey number inside the correct team.

Creating duplicate matches

If you want to update a fixture, reuse the same Match ID. Do not create a second match for the same game unless you really mean to.

A Practical Matchday Routine

If several organizers use the system, this routine keeps the dashboard clean.

Before Kickoff

Check the teams, player rosters, and fixture times. Make sure the match exists and is marked as scheduled.

After Full Time

Update the same match to played, save the final score, then add the player stats while the game is still fresh in your mind.

End Of The Day

Refresh the dashboard, check the highlights, publish the news if needed, and set the spotlight awards so the public page looks complete.
